1 步骤一：安装 Codex CLI

Terminal
node -v # 检查 Node 版本
npm i -g @openai/codex # 安装 Codex CLI
设置环境变量：
export OPENAI_BASE_URL="https://code.claude-opus.top/openai"
export OPENAI_API_KEY="你的API密钥"

永久设置 (zsh)：

echo 'export OPENAI_BASE_URL="https://code.claude-opus.top/openai"' >> ~/.zshrc
echo 'export OPENAI_API_KEY="你的API密钥"' >> ~/.zshrc
source ~/.zshrc

永久设置 (bash)：

echo 'export OPENAI_BASE_URL="https://code.claude-opus.top/openai"' >> ~/.bash_profile
echo 'export OPENAI_API_KEY="你的API密钥"' >> ~/.bash_profile
source ~/.bash_profile

2 步骤二：启动 Codex

codex # 定义你的模型

选择建议（强烈推荐选第1个）：

选项1（Full Access）
Codex可在/root目录直接执行命令或修改文件，无需逐次确认。效率最高。
选项2（Ask for Approval）
每次改文件/执行命令都会先询问确认，安全性更高，适合在生产机或关键目录操作。

3 步骤三：开始使用

进入 OpenAI Codex CLI 交互界面后即可开干。界面提供了几条常用命令：

/init
创建AGENTS.md配置模板
/status
查看当前会话配置
/approvals
设置操作确认模式
/model
选择模型和推理强度
建议配置：
gpt-5 + high；审批选择 Full Access，尽量减少打断

Codex 额外配置（推荐）

在 ~/.codex/config.toml 中添加：

disable_response_storage = true

这样可禁用响应存储，进一步提升隐私与安全。
