# 推荐设置速查

这页给新手一个保守、可执行的 Codex 设置参考。

## 默认建议

| 设置 | 推荐值 | 原因 |
| --- | --- | --- |
| 语言 | 中文 | 适合本教程站用户 |
| 推理深度 | Medium | 兼顾速度和稳定性 |
| 复杂任务 | High / Extra High | 需要跨文件分析时再升 |
| 文件权限 | Workspace write | 日常开发足够 |
| 网络权限 | 按需开启 | 查文档、安装依赖、部署时需要 |
| Plan Mode | 复杂任务开启 | 先确认范围再改 |

## AGENTS.md 推荐规则

```md
## Rules

- 默认用中文解释。
- 保留原有中转站、购买、飞书教程链接。
- 修改前先识别文件范围。
- 不要输出或写入任何密钥、token、cookie。
- 行为变化要补测试或给出验证方式。
```

## 什么时候提高权限

只有当任务明确需要时：

- 跨目录整理文件。
- 安装依赖。
- 部署到线上。
- 操作浏览器或外部应用。
- 读取非当前项目的素材。

提高权限前，先让 Codex 说明：

- 为什么需要。
- 会访问哪些路径。
- 会运行哪些命令。
- 如何避免泄露敏感信息。
