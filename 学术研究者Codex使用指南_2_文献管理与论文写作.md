## 第三章 写作项目环境配置

### 3.1 学术写作项目结构

建议采用以下文件夹结构组织论文写作项目：

```
paper-project/
├── manuscript/               # 论文主体
│   ├── main.tex             # 主文件
│   ├── sections/            # 各章节
│   │   ├── abstract.tex
│   │   ├── introduction.tex
│   │   ├── methods.tex
│   │   ├── results.tex
│   │   └── discussion.tex
│   └── bibliography.bib     # 参考文献
├── figures/                  # 图片文件
│   ├── raw/                 # 原始图片
│   └── processed/           # 处理后图片
├── tables/                   # 表格数据
├── supplementary/            # 补充材料
├── references/               # 文献PDF
│   └── pdfs/
└── notes/                    # 写作笔记
```

### 3.2 Codex配置优化

针对学术写作场景，推荐以下个性化规则：

```markdown
# 学术写作全局规则

## 写作规范
- 遵循学术写作规范，避免口语化表达
- 使用第三人称视角（除非期刊允许第一人称）
- 保持术语使用的一致性
- 段落之间保持逻辑连贯性

## LaTeX代码规范
- 遵循期刊投稿模板要求
- 使用有意义的标签命名（\label{fig:xxx}）
- 图片使用相对路径
- 表格使用booktabs宏包样式

## 参考文献规范
- 使用BibTeX格式管理文献
- 确保引用格式符合期刊要求
- DOI链接完整准确

## 输出要求
- 生成代码时提供编译说明
- 修改内容时标注具体位置
- 保持与现有文档风格一致
```

## 第三章 文献管理与信息提取

### 3.1 文献信息自动化处理

**场景**：批量提取PDF文献的元数据信息

**操作步骤**：

1. 创建新Thread，命名为"文献处理"
2. 描述文献处理需求

**示例提示词**：

```
请帮我处理 references/pdfs/ 目录下的PDF文献：

1. 提取每篇文献的以下信息：
   - 标题
   - 作者列表
   - 发表年份
   - 期刊/会议名称
   - DOI
   - 摘要

2. 生成BibTeX格式的参考文献条目
3. 创建文献信息汇总表格（CSV格式）
4. 按研究主题对文献进行分类

输出文件保存至 references/ 目录。
```

### 3.2 文献摘要生成

**场景**：为已阅读文献生成结构化摘要

**示例提示词**：

```
请为以下文献生成结构化摘要：

文献内容：[粘贴文献摘要或关键段落]

请按以下格式输出：
1. 研究背景（2-3句）
2. 研究目的（1-2句）
3. 研究方法（2-3句）
4. 主要发现（3-5句）
5. 研究局限（1-2句）
6. 对本研究的启示（2-3句）

字数控制在300字以内。
```

### 3.3 文献矩阵构建

**场景**：构建文献综述矩阵

**示例提示词**：

```
请根据以下文献列表构建文献综述矩阵：

文献1：[作者, 年份, 标题, 主要变量, 方法, 结论]
文献2：[...]
...

矩阵应包含以下列：
- 作者/年份
- 研究问题
- 理论框架
- 研究方法
- 样本特征
- 主要变量
- 核心发现
- 研究局限

输出为LaTeX表格格式，保存至 tables/literature_matrix.tex
```

## 第四章 论文框架与结构设计

### 4.1 使用Plan模式设计论文框架

**推荐工作流**：

```
论文写作 → Plan模式生成框架 → 审核调整 → 章节细化 → 内容填充
```

**示例提示词（IMRaD结构）**：

```
请帮我设计一篇关于[研究主题]的论文框架。

基本信息：
- 研究领域：[学科领域]
- 目标期刊：[期刊名称]
- 研究类型：[实证研究/综述/理论]
- 字数要求：[约X字]

请生成：
1. 论文整体结构（章节划分）
2. 每个章节的内容要点（3-5个要点）
3. 各章节的预估字数分配
4. 关键图表的规划建议

输出为详细的论文大纲文档。
```

### 4.2 各章节内容生成辅助

**引言章节示例**：

```
请帮我撰写论文引言部分的初稿。

研究背景信息：
- 研究领域：[领域]
- 研究缺口：[现有研究的不足]
- 研究问题：[核心问题]
- 研究意义：[理论与实践意义]

要求：
- 采用"漏斗式"结构：从一般到具体
- 包含3-5个关键文献的引用位置标注
- 字数约1000字
- 学术语气，避免过度主张
```

**方法章节示例**：

```
请帮我撰写研究方法章节。

研究设计信息：
- 研究类型：[实验/调查/案例研究等]
- 样本：[样本量、来源、特征]
- 数据收集：[方法、工具、过程]
- 数据分析：[统计方法、软件]
- 伦理审查：[批准信息]

要求：
- 描述详细，确保可复现性
- 符合[期刊名称]的格式要求
- 包含必要的技术细节
```

### 4.3 讨论章节结构化

**示例提示词**：

```
请帮我组织讨论章节的结构。

研究发现摘要：
- 发现1：[描述]
- 发现2：[描述]
- 发现3：[描述]

请生成讨论章节框架：
1. 主要发现总结（与假设/研究问题对照）
2. 与现有文献的对话（支持/不一致）
3. 理论贡献
4. 实践启示
5. 研究局限
6. 未来研究方向

每个部分提供3-5个讨论要点。
```


### 5.2 表格生成与美化

**场景**：生成论文级统计结果表格

**示例提示词**：

```
请将以下统计结果转换为LaTeX表格：

数据：
| 变量 | M | SD | 1 | 2 | 3 |
|------|------|------|------|------|------|
| 变量1 | 3.45 | 0.78 | - | | |
| 变量2 | 4.12 | 0.92 | .45** | - | |
| 变量3 | 2.89 | 1.05 | .32* | .28* | - |

要求：
- 使用 booktabs 宏包样式
- 包含表格注释（*p<.05, **p<.01）
- 符合APA格式
- 表格宽度适应双栏排版
```

**生成代码示例**：

```latex
\begin{table}[htbp]
\centering
\caption{变量描述性统计与相关系数矩阵}
\label{tab:correlation}
\begin{threeparttable}
\small
\begin{tabular}{lccccc}
\toprule
变量 & $M$ & $SD$ & 1 & 2 & 3 \\
\midrule
1. 变量1 & 3.45 & 0.78 & -- & & \\
2. 变量2 & 4.12 & 0.92 & .45** & -- & \\
3. 变量3 & 2.89 & 1.05 & .32* & .28* & -- \\
\bottomrule
\end{tabular}
\begin{tablenotes}
\small
\item 注：* $p < .05$, ** $p < .01$.
\end{tablenotes}
\end{threeparttable}
\end{table}
```

### 5.3 数学公式排版

**场景**：生成规范的数学公式

**示例提示词**：

```
请将以下统计模型转换为LaTeX公式：

模型描述：
- 因变量：Y（连续变量）
- 自变量：X1, X2, X3
- 控制变量：C1, C2
- 层次结构：个体嵌套于组

请生成：
1. 完整的多层线性模型公式
2. 各参数的说明表格
3. 使用 align 环境排版
```

### 5.4 参考文献格式配置

**常见期刊引用格式配置**：

| 期刊类型 | BibTeX样式 | 说明 |
|----------|------------|------|
| APA格式 | apacite | 心理学、社会科学 |
| IEEE格式 | IEEEtran | 工程技术 |
| Elsevier | elsarticle-harv | 多数Elsevier期刊 |
| Springer | spbasic | Springer期刊 |

**配置示例**：

```latex
% APA格式
\usepackage{apacite}
\bibliographystyle{apacite}

% 引用方式
\cite{Author2024}        % 单引
\citeA{Author2024}       % 作为句子主语
\citeyear{Author2024}    % 仅年份
```

## 第八章 质量控制与投稿准备

### 8.1 投稿前检查清单

| 检查项目 | 说明 | 状态 |
|----------|------|------|
| 标题字数 | 符合期刊限制 | □ |
| 摘要结构 | 符合期刊要求 | □ |
| 关键词数量 | 通常4-6个 | □ |
| 字数限制 | 符合期刊规定 | □ |
| 图表质量 | ≥300 DPI | □ |
| 参考文献格式 | 符合期刊样式 | □ |
| 引用完整性 | 文内引用与文献列表一致 | □ |
| 利益冲突声明 | 按要求填写 | □ |
| 数据可用性声明 | 按要求填写 | □ |

### 8.2 常见格式问题解决

| 问题 | 解决方案 |
|------|----------|
| 图表浮动位置不当 | 使用 [htbp] 或 [H] 选项 |
| 参考文献格式错误 | 检查 .bib 文件格式 |
| 中文显示异常 | 确认 ctex 宏包加载 |
| 公式编号错误 | 检查 equation 环境嵌套 |
| 表格超出页边距 | 使用 tabularx 或调整列宽 |

### 8.3 版本控制最佳实践

```
- [写作] 内容撰写/修改

- [图表] 图表更新
```
