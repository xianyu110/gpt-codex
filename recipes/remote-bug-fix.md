# Codex x 云服务器: 远程定位并修复 Bug

## 适合谁

适合有服务器、服务日志、部署脚本的团队。

## 准备输入

- 服务器访问方式。
- 服务名。
- 错误日志。
- 部署流程。

## 提示词

```text
请帮我定位线上服务 500 错误。
要求：先只读查看日志和服务状态；不要重启、删除、改配置，直到给出计划并确认。
```

## Codex 应该做什么

- 查看状态。
- 查日志。
- 判断是否是配置、依赖、代码或资源问题。
- 给出最小修复。
- 确认后执行。

## 验证

- 服务恢复。
- 错误日志消失。
- 有回滚方案。

## 风险

- 误删生产数据。
- 重启影响用户。
- 把服务器密钥写入日志。
