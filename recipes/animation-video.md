# Codex x 动画视频: 用代码生成动画

## 适合谁

适合想把概念、产品流程、教程步骤做成短视频或网页动画的人。

## 准备输入

- 文案脚本。
- 视频比例。
- 时长。
- 品牌色和字体要求。

## 提示词

```text
请用代码生成一个 30 秒竖屏动画，主题是 Codex 从 README 生成网页。
要求：字幕清晰，节奏分 4 幕，输出可预览 HTML 和最终 MP4 的生成命令。
```

## Codex 应该做什么

- 拆分分镜。
- 生成动画代码。
- 本地预览。
- 导出视频或给出导出命令。

## 验证

- 播放不卡顿。
- 字幕不溢出。
- 画面比例正确。

## 风险

- 只做了静态图。
- 字幕太小。
- 导出依赖没有说明。
