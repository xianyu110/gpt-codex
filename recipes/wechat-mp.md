# Codex x 公众号: 内容采集与发布流程

## 适合谁

适合公众号运营，把选题、采集、改写、排版、草稿上传串起来。

## 准备输入

- 选题来源。
- 公众号风格要求。
- 图片素材。
- 是否具备上传脚本和环境变量。

## 提示词

```text
请把这篇素材整理成公众号草稿。
要求：保留事实来源；生成标题、摘要、正文、封面图提示词；
没有 WECHAT_APPID 和 WECHAT_APPSECRET 时只生成 Markdown，不上传。
```

## Codex 应该做什么

- 提取素材。
- 改写为公众号结构。
- 处理图片。
- 检查环境变量。
- 有条件再上传草稿。

## 验证

- Markdown 可发布。
- 图片链接可访问。
- 没有凭据泄露。
- 上传失败有日志。

## 风险

- 版权和来源不清。
- 凭据泄露。
- 未确认就发布。
