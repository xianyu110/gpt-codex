# Codex x Chrome: 直接控制浏览器

## 适合谁

适合需要使用本机 Chrome 登录态的网站操作，比如后台管理、资料下载、网页检查。

## 准备输入

- Chrome 已登录目标网站。
- 明确操作步骤。
- 哪些动作需要确认。

## 提示词

```text
请使用 Chrome 打开后台页面，检查最近 10 条草稿标题。
要求：只读查看，不发布、不删除、不修改；结果整理成表格。
```

## Codex 应该做什么

- 使用现有 Chrome 状态。
- 打开指定页面。
- 读取可见数据。
- 不做写操作。

## 验证

- 数据和页面一致。
- 没有误点击发布/删除。
- 敏感信息不输出。

## 风险

- 误操作真实后台。
- 登录态过期。
- 页面内容涉及隐私。
