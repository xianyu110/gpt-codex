# Codex x 安卓手机: 扫码连接，远程操控

## 适合谁

适合需要在安卓手机上做测试、截图、重复操作的人。

## 准备输入

- 手机连接方式。
- ADB 或投屏工具。
- 操作步骤。
- 测试目标。

## 提示词

```text
请通过 ADB 检查安卓 App 的登录页。
要求：只做截图和页面跳转，不输入真实账号密码；保存截图并汇报控件状态。
```

## Codex 应该做什么

- 检查设备连接。
- 截图。
- 执行安全点击。
- 输出结果。

## 验证

- 设备在线。
- 截图存在。
- 没有误发消息或误支付。

## 风险

- 操作真实账号。
- 误触支付或发送。
- 设备权限不清。
