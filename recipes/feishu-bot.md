# Codex x 飞书: 一句话处理多维表格 / 机器人

## 适合谁

适合团队把飞书表格、文档、机器人和日程接入 Codex。

## 准备输入

- 飞书应用权限。
- 目标文档或多维表格。
- 操作范围。
- 是否允许发送消息。

## 提示词

```text
请读取这个飞书多维表格，统计本周新增线索数量。
要求：只读统计；不要修改字段；输出中文摘要。
```

## Codex 应该做什么

- 检查权限。
- 读取目标表。
- 统计数据。
- 需要写入或发消息前确认。

## 验证

- 统计数量和飞书表格一致。
- 没有修改原表。
- 权限范围最小。

## 风险

- 应用权限不足。
- 误写表格。
- 群消息误发。
