# Codex x Playwright MCP: 让 AI 操控浏览器

## 适合谁

适合要测试网页、填表、截图、检查交互的人。

## 准备输入

- 要访问的 URL。
- 测试目标。
- 登录方式。
- 期望截图或结果。

## 提示词

```text
请用 Playwright 打开本地页面 http://127.0.0.1:8765，
检查首页、docs 入口和 recipes 入口是否能点击。
要求：截图保存到 artifacts/，并汇报坏链接。
```

## Codex 应该做什么

- 打开浏览器。
- 点击关键入口。
- 保存截图。
- 汇报错误。

## 验证

- 截图存在。
- 控制台没有明显错误。
- 链接可以打开。

## 风险

- 登录态不可用。
- 页面动态加载太慢。
- 只看 DOM 不看实际截图。
