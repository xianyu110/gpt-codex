# Codex x README: 把文档一键变网页

## 适合谁

适合第一次想看到 Codex 真实产出的用户。这个案例能练到读文件、改 HTML、保留链接、本地预览和检查 diff。

## 准备输入

- 一个 `README.md`
- 一个空白或已有的 `index.html`
- 原有购买、中转、教程链接清单

## 提示词

```text
请把 README.md 的核心内容整理成一个静态首页。
要求：保留所有原有中转站、购买、飞书教程链接；首页只放入口、学习路线和案例入口；
长教程移动到 docs/guide，实战放到 recipes。完成后用本地预览检查。
```

## Codex 应该做什么

- 搜索 README 里的链接。
- 规划首页结构。
- 修改 `index.html` 或生成新的静态页面。
- 添加 `docs/` 和 `recipes/` 入口。
- 启动本地预览。
- 汇报保留的关键链接。

## 验证

```bash
python3 -m http.server 8765
```

打开 `http://127.0.0.1:8765/`。

检查：

- 首页能打开。
- 国内站入口还在。
- 购买套餐入口还在。
- 飞书教程入口还在。
- 能跳到 `docs/` 和 `recipes/`。

## 风险

- 把原有转化链接删掉。
- 首页塞太长，变成文章页。
- 没有移动端适配。
