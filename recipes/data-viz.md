# Codex x 数据可视化: CSV 变图表

## 适合谁

适合有 CSV、Excel 或导出数据，想快速得到图表和结论的人。

## 准备输入

- CSV 或 XLSX 文件。
- 想回答的问题。
- 输出格式：PNG、HTML、Markdown 报告。

## 提示词

```text
请读取 data.csv，分析最近 30 天的增长趋势。
要求：先检查字段含义和缺失值；生成 3 张图和一段结论；
不要修改原始数据，输出到 reports/。
```

## Codex 应该做什么

- 检查数据结构。
- 清洗缺失值。
- 生成图表。
- 写出结论和限制。

## 验证

- 图表能打开。
- 结论能对应数据。
- 原始数据没有被覆盖。

## 风险

- 字段理解错。
- 图表没有标单位。
- 样本太小却下结论过重。
