# Codex x 资料调研: 网络信息整理成报告

## 适合谁

适合做竞品调研、市场分析、技术选型和资料汇总的人。

## 准备输入

- 调研问题。
- 时间范围。
- 可信来源要求。
- 报告结构。

## 提示词

```text
请调研 Codex、Claude Code 和其他 coding agent 的入门路径。
要求：优先官方来源；每条关键事实附链接；输出中文报告和证据表。
```

## Codex 应该做什么

- 搜索可信来源。
- 对比信息。
- 标注时间。
- 写报告。
- 附证据表。

## 验证

- 链接可打开。
- 日期清楚。
- 事实和观点分开。

## 风险

- 使用过时资料。
- 来源不可验证。
- 没区分官方信息和个人观点。
