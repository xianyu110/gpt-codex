# Codex x 文献综述: 整理成可复核证据表

## 适合谁

适合学生、研究者、产品经理做论文或资料综述。

## 准备输入

- 论文 PDF 或链接。
- 研究问题。
- 输出格式。
- 引用要求。

## 提示词

```text
请阅读这些论文，围绕“AI 编程助手对开发效率的影响”整理文献综述。
要求：每条结论必须对应论文来源；输出证据表、主题归纳和未解决问题。
```

## Codex 应该做什么

- 提取论文元数据。
- 总结方法和结论。
- 建证据表。
- 标注来源。

## 验证

- 每条结论可追溯。
- 没有编造引用。
- 研究问题没有跑偏。

## 风险

- 摘要替代全文。
- 引用格式混乱。
- 把相关性写成因果。
