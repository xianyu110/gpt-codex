# Codex x Notion MCP: 打通知识空间

## 适合谁

适合已经在 Notion 里维护项目、知识库或任务表的人。

## 准备输入

- Notion MCP 配置。
- 目标数据库或页面。
- 读写权限范围。
- 任务目标。

## 提示词

```text
请通过 Notion MCP 读取指定数据库，整理最近一周未完成任务。
要求：只读取任务标题、负责人、截止时间和状态；先汇总，不要写入。
```

## Codex 应该做什么

- 检查 MCP 是否可用。
- 读取授权范围内的数据。
- 汇总结果。
- 写入前请求确认。

## 验证

- 结果数量和 Notion 页面一致。
- 没有读取无关数据库。
- 写操作有确认。

## 风险

- 权限过大。
- 误改页面。
- 把私人内容输出到日志。
