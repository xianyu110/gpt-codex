# Codex x 数据库 MCP: 自然语言查数据

## 适合谁

适合运营、产品、开发想用自然语言查询数据库，但又不想直接暴露生产权限的人。

## 准备输入

- 数据库 MCP。
- 只读账号。
- 表结构说明。
- 查询目标。

## 提示词

```text
请通过数据库 MCP 查询过去 7 天注册用户数。
要求：只读查询；先解释要查哪些表和字段；不要导出个人敏感信息。
```

## Codex 应该做什么

- 理解表结构。
- 写只读 SQL。
- 执行查询。
- 输出汇总。

## 验证

- SQL 可复核。
- 查询范围正确。
- 不包含敏感字段。

## 风险

- 查询生产库压力太大。
- 泄露个人信息。
- 指标口径没讲清。
