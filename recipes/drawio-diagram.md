# Codex x Draw.io: 自动绘制架构图

## 适合谁

适合需要把系统、业务流程、部署架构画成可编辑图的人。

## 准备输入

- 系统说明。
- 模块列表。
- 数据流。
- 想要的图类型：流程图、架构图、泳道图。

## 提示词

```text
请根据 README 和 docs 里的系统说明生成 Draw.io 架构图。
要求：输出 .drawio 文件；包含前端、API、数据库、对象存储和通知服务；
模块之间标注数据流，不要只给图片。
```

## Codex 应该做什么

- 读取系统说明。
- 提炼节点和边。
- 生成 `.drawio` 或 XML。
- 提供打开和检查方式。

## 验证

- 用 draw.io 打开文件。
- 节点和关系可编辑。
- 模块命名与文档一致。

## 风险

- 凭空添加不存在的服务。
- 图能看但不可编辑。
- 数据流方向画反。
