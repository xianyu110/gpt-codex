# Codex x 定时巡检: 服务器自动巡检 + 通知

## 适合谁

适合需要每天检查服务器、服务状态、磁盘、日志和告警的人。

## 准备输入

- 服务器列表。
- 检查项。
- 通知渠道。
- 阈值规则。

## 提示词

```text
请设计一个每日服务器巡检自动化。
检查项：磁盘、内存、CPU、服务状态、错误日志。
要求：只读检查；异常才通知；日志保存到 reports/。
```

## Codex 应该做什么

- 写巡检脚本。
- 配置阈值。
- 生成报告。
- 配置通知。
- 记录失败。

## 验证

- 手动运行成功。
- 报告可读。
- 异常规则准确。

## 风险

- 通知刷屏。
- 权限过大。
- 日志里含敏感信息。
