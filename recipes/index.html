Codex 实战案例库

入口页解决“怎么买、怎么进、怎么配置”，案例库解决“进来以后能做什么”。这里把 Codex x X 的工作流拆成 24 个可复用案例，每个案例都有准备输入、提示词、执行动作、验证方式和风险提醒。

先做 README 变网页 查看全部案例 第一个任务教程

学习顺序

  • 先做 README 变网页，建立改文件和验证的手感。
  • 再做静态站部署，把结果发出去。
  • 最后按自己的工作场景选 PPT、Obsidian、飞书、CI 或自动化。
