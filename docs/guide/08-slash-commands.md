# 08. 斜杠命令 + 状态

斜杠命令是 Codex 里切换模式和查看状态的快捷方式。不同版本和界面可能略有差异，以当前 App 内展示为准。

## 常用命令

| 命令 | 作用 | 使用时机 |
| --- | --- | --- |
| `/plan` | 进入或切换规划模式 | 复杂任务开始前 |
| `/status` | 查看当前状态、上下文或用量 | 长任务中途 |
| `/model` | 切换模型或查看模型 | 需要更强或更快模型 |
| `/help` | 查看可用命令 | 不确定命令时 |

## 新手最常用组合

```text
/plan
请先检查仓库结构，列出教程改造计划。
```

执行中途：

```text
/status
```

收尾时：

```text
请总结改动文件、验证命令和剩余风险。
```

## 注意

- 不同 Codex surface 的命令可能不同。
- 看到不确定的命令，优先用 `/help` 查当前环境。
- 不要为了“显得专业”乱切模式，能完成任务最重要。
