# 02. 订阅 ChatGPT Plus / Pro

Codex 的可用额度、模型和部分云端能力与账号套餐有关。新手不要先背所有套餐细节，只需要知道自己要的是“官方订阅体验”还是“第三方中转快速跑通”。

## 官方订阅适合谁

官方订阅适合：

- 想使用官方原生体验的人。
- 可以稳定访问 ChatGPT 和 Codex 的人。
- 能处理支付方式、地区和账号问题的人。
- 需要更完整 ChatGPT 功能的人。

## Plus / Pro 怎么理解

| 套餐 | 适合场景 | 建议 |
| --- | --- | --- |
| Free / Go | 轻量体验、测试能不能用 | 先跑小任务 |
| Plus | 每周几次认真使用 Codex | 新手首选 |
| Pro | 高频开发、长任务、多线程 | 等你真的用起来再升 |
| Business / Enterprise / Edu | 团队、学校、企业 | 看组织管理和合规要求 |

模型和额度会随官方策略变化，最终以你账号内展示为准。

## 订阅前检查

订阅前先确认：

- 你能正常登录 ChatGPT。
- 你能打开 Codex 页面。
- 你知道自己要用桌面 App、CLI 还是 Web。
- 你接受官方订阅对网络、支付和账号环境的要求。

## 和第三方中转的关系

官方订阅不是第三方中转。第三方中转的价值是降低访问、支付和配置门槛，适合想快速跑通的人。
